# Simple ATM System

A simple **C++ console-based ATM system** built as a **training project**.

## Purpose
This project is for learning and practice only.  
I built it to improve my understanding of:
- Structs and enums in C++
- File handling (`fstream`)
- Menu-driven console applications
- Basic transaction logic (deposit, withdraw, balance check)
- Login validation using account number and pin code

## Features
- Login with account number and pin code
- Quick withdraw
- Normal withdraw
- Deposit
- Check balance
- Save and load client data from `Clients.txt`

## Notes
- This is a **beginner-level educational project**.
- It is not production-ready and does not include advanced security features.
- Future improvements are welcome.
